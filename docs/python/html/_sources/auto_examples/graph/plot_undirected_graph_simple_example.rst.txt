Undirected Graph: Simple Example
=============================================

Very simple example how to use undirected graphs




.. code-block:: python

    from __future__ import print_function
    import nifty.graph
    import numpy
    import pylab







2D undirected grid graph



.. code-block:: python

    numberOfNodes = 5
    graph = nifty.graph.undirectedGraph(numberOfNodes)
    print("#nodes", graph.numberOfNodes)
    print("#edges", graph.numberOfEdges)
    print(graph)





.. rst-class:: sphx-glr-script-out

 Out::

    #nodes 5
    #edges 0
    #Nodes 5 #Edges 0


insert edges



.. code-block:: python

    graph.insertEdge(0,1)
    graph.insertEdge(0,2)










insert multiple edges at once



.. code-block:: python

    uvIds = numpy.array([[0,3],[1,2],[1,4],[1,3],[3,4]])
    graph.insertEdges(uvIds)






iterate over nodes
and the adjacency
of each node



.. code-block:: python

    for node in graph.nodes():
        print("u",node)
        for v,e in graph.nodeAdjacency(node):
            print(" v",v,"e",e)

 Out::

iterate over edges
and print the endpoints



.. code-block:: python

    for edge in graph.edges():
        print("edge ",edge, "uv:", graph.uv(edge))






.. rst-class:: sphx-glr-script-out

 Out::

    edge  0 uv: (0, 1)

    edge  3 uv: (1, 2)
    edge  4 uv: (1, 4)
    edge  5 uv: (1, 3)
    edge  6 uv: (3, 4)


get the uv-ids /endpoints
for all edges simultaneous
as a numpy array



.. code-block:: python

    uvIds = graph.uvIds()
    print(uvIds)

 Out::

plot the graph:
needs networkx



.. code-block:: python

    nifty.graph.drawGraph(graph)
    pylab.show()
