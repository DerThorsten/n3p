template<class G, class T>
struct nifty::graph::graph_maps::MultibandNodeMap< G, T >

Multiband node map.

Sometimes we need to hold not a single scalar, but a fixed length vector for each node. The return type of operator[] is a tiny proxy object holding the vector.
