template<class OBJECTIVE, class ILP_SOLVER>

ILP Cutting plane multicut solver.

ILP Cutting plane multicut solver adding violated cycle constraints in an iterative fashion.

Inform solver about a change of weights.

Inform solver that all weights could have changed. If a particular solver does not overload this function, an WeightsChangedNotSupported exception is thrown. After a call of this function it is save to run optimize again, therefore it resets the solver
